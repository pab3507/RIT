Terminology

  • Internet - A computer network consisting of a worldwide network of computer networks that use the TCP/IP network protocols to facilitate data transmission and exchange.
  • World Wide Web - an information system on the Internet that allows documents to be connected to other documents by hypertext links, enabling the user to search for information by moving from one document to another.
  • Multimedia - Transmissions that combine media of communication (text and graphics and sound etc.)
  • Hypertext - Machine-readable text that is not sequential but is organized so that related items of information are connected.
  • CLI – A Command Line Interface is a user interface in which you type commands instead of choosing them from a menu or selecting an icon.
  • GUI - A Graphical User Interface is a user interface based on graphics (icons and pictures and menus) instead of text; uses a mouse as well as a keyboard as an input device.
  • HCI - Human–computer interaction researches the design and use of computer technology, focused on the interfaces between people (users) and computers.
  • FTP – The File Transfer Protocol allows users to copy files between their local system and any system they can reach on the network; verb use the file transfer protocol to transfer data from one computer to another.
  • SSH – Secure Shell is a cryptographic network protocol for operating network services securely over an unsecured network.
  • HTTP – The Hypertext Transfer Protocol allows the transfer of hypertext requests and information between servers and browsers.
  • Telnet - A network protocol that allows a user on one computer to log onto another computer that is part of the same network.
  • HTML - Hypertext Markup Language, a standardized system for tagging text files to achieve font, color, graphic, and hyperlink effects on World Wide Web pages.
